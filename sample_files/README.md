# 샘플 파일 폴더

이 폴더는 테스트용 샘플 파일들을 저장하는 곳입니다.

## 사용법

1. 이 폴더에 `sample.pdf` 파일을 추가하세요
2. `genos_di/test.py`를 실행하면 해당 파일을 처리합니다

## 지원되는 파일 형식

- PDF (.pdf)
- HWP (.hwp) 
- DOCX (.docx)
- TXT (.txt)
- 기타 DocumentProcessor가 지원하는 형식

## 파일명 규칙

- 기본적으로 `sample.pdf`를 찾습니다
- 다른 파일명을 사용하려면 `test.py`의 `file_path` 변수를 수정하세요
